# jcs-linux-lab

A personal Oracle Linux lab environment for learning enterprise Linux, VirtualBox automation, LVM, networking, SELinux, firewalld, and system administration.

## Contents
- Automated VM creation scripts
- LVM practice scripts
- Networking configuration examples
- Guest Additions setup
- Troubleshooting notes
- Architecture diagrams

## Goals
- Build repeatable Oracle Linux VMs
- Practice enterprise Linux administration
- Learn LVM, SELinux, firewalld, networking
- Create a reusable automation toolkit

## Requirements
- VirtualBox
- Oracle Linux 10.1 ISO
- Bash (for scripts)
